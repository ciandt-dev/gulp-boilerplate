Why start a project using Gulp Boilerplate?

It's simple, with this feature lets you optimize the time you lose in compiling files, minify and even mounting images to sprite.

To start a project, you just follow a few simple steps. Don't worry my friend, the steps are exactly the same for both Linux and Windows.

You can help this project evolve more and more. Yup! We will accept your contribution, feel avontade to create commits or issues. In fact, we created a special documentation for this.

I can use in any project?

Yep! This can be used in any application, without any problem, it's simple to use and set up, you can follow the steps of the configuration folder and run the Gulp and ready!
